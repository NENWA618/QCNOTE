# NOTE - 快速开始指南

欢迎使用 NOTE！一个简洁而优雅的个人笔记应用。本指南将帮助你快速上手。

## 🚀 5 分钟快速开始

### 前置要求

- **Node.js** 18+ 和 npm（Node 包管理器）
- 任何代码编辑器（推荐 VS Code）

### 步骤 1：安装依赖

```bash
npm install
```

### 步骤 2：启动开发服务器

```bash
npm run dev
```

服务器将启动在 `http://localhost:3003`（或下一个可用端口）。

### 步骤 3：在浏览器打开

打开以下地址：

| 页面 | URL                             | 说明               |
| ---- | ------------------------------- | ------------------ |
| 首页 | http://localhost:3003           | 应用首页和功能介绍 |
| 笔记 | http://localhost:3003/dashboard | 创建、管理笔记     |
| 支持 | http://localhost:3003/contact   | 支持我们的页面     |

## 📝 基本操作

### 笔记管理

| 操作     | 方法                     | 快捷键   |
| -------- | ------------------------ | -------- |
| 新建笔记 | 点击"➕ 新建笔记"按钮    | -        |
| 搜索笔记 | 在搜索框输入关键词       | `Ctrl+K` |
| 编辑笔记 | 点击笔记卡片或"编辑"按钮 | -        |
| 删除笔记 | 点击"删除"按钮           | -        |
| 更改分类 | 侧边栏选择分类           | -        |
| 排序笔记 | 侧边栏选择排序方式       | -        |

### 笔记编辑

在编辑面板中：

1. **标题**：输入笔记标题
2. **内容**：编写笔记正文
3. **分类**：选择分类（生活、工作、学习、灵感）
4. **标签**：添加多个标签（用逗号分隔）
5. **保存**：点击"✓ 保存"按钮

## 🎨 笔记特性

### 分类系统

- **生活** - 日常生活笔记
- **工作** - 工作相关内容
- **学习** - 学习笔记
- **灵感** - 创意和想法

### 统计功能

侧边栏实时显示：

- 总笔记数
- 各分类笔记数
- 收藏数

## 💾 数据存储

✅ **完全本地存储**：所有笔记保存在浏览器的本地存储中  
✅ **100% 隐私保护**：数据永不上传到服务器  
✅ **跨浏览器标签页同步**：在同一浏览器中自动同步

> 💡 **提示**：清除浏览器缓存会删除所有本地笔记。建议定期导出重要笔记。

## ❓ 常见问题

**Q: 我的笔记会保存在哪里？**  
A: 所有笔记保存在浏览器的本地存储中（localStorage），不会上传到任何服务器。

**Q: 我可以导出我的笔记吗？**  
A: 目前还未实现导出功能。我们计划在未来版本支持导出为 JSON、Markdown 等格式。

**Q: 可以在多个设备上同步笔记吗？**  
A: 目前不支持。笔记只在单个浏览器中保存。

**Q: 如何恢复已删除的笔记？**  
A: 删除后无法恢复。请谨慎删除重要笔记。
