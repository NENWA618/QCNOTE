# 🎉 QCNOTE - 你的智能笔记伙伴

**让记录变得简单而优雅。**

QCNOTE 是一款智能个人笔记应用，支持离线优先、AI 增强和多平台同步。

## 🚀 快速开始

### 本地运行
    git clone https://github.com/NENWA618/NOTE.git
    cd NOTE
    npm install
    npm run dev

### 在线体验
访问 [qcnote.com](https://qcnote.com) 立即开始使用。

## ✨ 核心特性

- **Markdown 编辑**：完整语法支持，实时预览
- **智能搜索**：全文与语义搜索
- **暗黑模式**：护眼模式，支持主题切换
- **AI 助手**：标签生成、摘要、情感分析
- **多端同步**：支持 WebDAV、OneDrive、iCloud
