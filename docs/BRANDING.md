# 品牌说明：QCNOTE 与 NOTE

本文件用于明确区分项目中 `QCNOTE` 与 `NOTE` 的含义，避免命名混淆。

## QCNOTE

- 这是当前项目的正式品牌名。
- 用于应用名称、页面标题、UI 文案、扩展名称、服务名称、存储键、测试用例、部署配置等。
- 代表当前仓库中的产品，已替换为全局品牌标识。
- 例如：`QCNOTE.code-workspace`、`QCNOTE-server`、`QCNOTE Web Clipper`、`QCNOTE_STORAGE` 等。

## NOTE

- 原始项目名称或通用“笔记”词语。
- 用于保留的外部仓库链接、开源来源引用、技术性注释以及不应替换的历史上下文。
- 例如：`https://github.com/NENWA618/NOTE`、`NOTE:` 注释前缀、原始项目仓库引用等。
- NOTE 也可作为普通英文单词“笔记”的通用表达，不一定代表品牌。

## 说明

- 绝大多数品牌相关内容已统一替换为 `QCNOTE`。
- 对于外部链接和原始仓库引用，仍保留 `NOTE` 以保持来源关系。
- `QCNOTE` 代表当前项目的品牌身份；`NOTE` 则代表原始来源、旧版本或通用名词。
