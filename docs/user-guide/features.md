# ✨ QCNOTE 功能特性详解

探索 QCNOTE 的强大功能，让笔记管理变得更加高效和有趣。

---

## 📝 核心功能

### 智能笔记编辑

#### Markdown 支持
- **完整语法支持**：标题、列表、链接、图片、表格、代码块
- **实时预览**：边写边看效果
- **语法高亮**：代码块支持多种编程语言
- **数学公式**：LaTeX 数学表达式支持

#### 自动保存
- **实时保存**：每30秒自动保存，永不丢失
- **版本历史**：查看和恢复笔记的历史版本
- **草稿管理**：自动保存未完成的草稿

#### 富媒体支持
- **图片上传**：支持拖拽上传和粘贴图片
- **文件附件**：添加文档、音频、视频等附件
- **链接预览**：自动生成网页链接的预览卡片

## 🧠 AI 增强功能

### 智能标签生成
- **自动分析**：AI 分析笔记内容，自动生成相关标签
- **多语言支持**：支持中英文标签生成
- **自定义标签**：可以手动调整和添加标签

### 内容摘要
- **智能摘要**：一键生成笔记的核心要点
- **长度控制**：可以选择摘要的详细程度
- **多格式输出**：支持 bullet points、段落等多种格式

### 情感分析
- **情感识别**：分析笔记内容的情感倾向
- **情绪追踪**：记录写作时的情绪变化
- **洞察报告**：生成个人情绪分析报告

### 智能推荐
- **相关笔记**：自动推荐相关内容的笔记
- **阅读建议**：基于你的阅读习惯推荐内容
- **写作提示**：AI 助手提供写作建议

## 🔍 搜索与发现

### 全文搜索
- **实时搜索**：输入即搜索，无需点击确认
- **模糊匹配**：支持拼写错误和近似匹配
- **高亮显示**：搜索结果中的关键词高亮显示

### 向量搜索
- **语义搜索**：理解你的意图而不仅是关键词
- **智能排序**：按相关度排序搜索结果
- **上下文理解**：考虑笔记之间的关联关系

### 高级筛选
- **时间筛选**：按创建时间、修改时间筛选
- **分类筛选**：按笔记分类快速筛选
- **标签筛选**：支持多标签组合筛选
- **状态筛选**：收藏、草稿、已发布等状态
