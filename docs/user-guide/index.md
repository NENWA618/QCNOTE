## 💡 使用技巧

### 快速编辑技巧
- 使用 Markdown 语法快速格式化
- 支持代码段高亮显示
- 使用数学公式支持 LaTeX
- 嵌入图片、视频和表格

### 高效搜索
- 使用 AND/OR 逻辑操作符
- 按标签和时间范围筛选
- 利用语义搜索理解内容意思
- 保存常用搜索作为快速查询

### 性能优化
- 定期清理旧版本和临时数据
- 使用本地搜索而不是全文索引
- 启用增量同步减少流量
- 设置自动备份频率
